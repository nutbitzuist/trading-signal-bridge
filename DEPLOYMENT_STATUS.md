The Railway CLI requires interactive terminal input. Please follow these steps:

### Step 1: Navigate to Backend Directory
```bash
cd "/Users/nut/Desktop/TV to MT connection/trading-signal-bridge/backend"
```

### Step 2: Initialize Railway Project
```bash
railway init
```
- Choose workspace: **nutbitzuist's Projects**
- Project name: **trading-signal-bridge-backend**

### Step 3: Add PostgreSQL
```bash
railway add --database postgresql
```

### Step 4: Add Redis
```bash
railway add --database redis
```

### Step 5: Set Environment Variables
```bash
# Generate secure keys
railway variables set SECRET_KEY="$(openssl rand -hex 32)"
railway variables set WEBHOOK_SECRET="$(openssl rand -hex 32)"

# Set CORS to allow your Vercel frontend
railway variables set CORS_ORIGINS="https://trading-signal-bridge-nutbitzuists-projects.vercel.app,https://trading-signal-bridge-ama804f8m-nutbitzuists-projects.vercel.app,http://localhost:3000"
```

### Step 6: Deploy
```bash
railway up
```

### Step 7: Create Public Domain
```bash
railway domain create
```

This will give you a URL like: `https://trading-signal-bridge-backend-production.up.railway.app`
